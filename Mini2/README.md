# Know Your Audience: A Closer Insight on Pinterest Women's Users

## Motivation

In every project we do, we need to know who we are creating for. Whether it is a product, a marketing campaign, or a creative project, understanding your audience is important. Pinterest is one of the largest platforms for sharing inspiration, fashion, beauty, and lifestyle ideas, making it a great place to study user interests and behavior.
Also, it is my dream to work at Pinterest.
The goal of this project was to explore who uses Pinterest, where they are located, and how interests differ among female users of different age groups. This project is also built upon the data collected last semester for a data wrangeling class. 

## Data Description

This project combines three datasets:

1. Pinterest users by country (2025), used to analyze the global distribution of Pinterest users scraped from the statista.
2. Gender distribution data from Statista, used to examine the ratio of female and male Pinterest users.
3. An enriched Pinterest trends dataset containing search trends for female users aged 18–49. This dataset was processed and enriched using a large language model to categorize trends into themes and make the data easier to analyze.

Several data cleaning steps were performed, including handling missing values, filtering records without search volume, and replacing missing classifications with default categories.

## Visualizations

The project includes:

* An interactive world map showing Pinterest users by country.
* A gender distribution visualization highlighting Pinterest's predominantly female audience.
* Interactive user cards showing differences between female age groups.
* A modeled trend analysis using linear regression to examine how search interest changes over time.

## Key Findings

The United States has the largest Pinterest audience, surprisingly followed by several rapidly growing international markets such as Brazil.

Pinterest is primarily used by women, who represent the majority of the platform's user base.

Female users aged 18–24 showed the greatest diversity of interests and the highest number of unique trends. Older age groups generally focused on a smaller number of recurring themes, primarily beauty and makeup.

The trend analysis showed that most categories experienced a decline in search volume throughout the observation period, while nail-related content was the only category that showed positive growth. The results also revealed seasonal patterns and changing aesthetics that influence how users search for inspiration online.

## Summary

This project demonstrates how data visualization can be used to transform raw search data into meaningful stories about real people. By combining geographic, demographic, and behavioral data, the analysis provides insight into who Pinterest users are, what inspires them, and how trends evolve over time.
