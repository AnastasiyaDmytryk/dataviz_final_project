## Mini-Project 1

This mini project is inspired by the Starbucks data set and it goes into the dietary values of drinks
The mini project creates 3 visualizations on : 
Top 5 drinks with more sugar than caffeine
Top 5 healthiest drinks 
and
Top 5 worst drinks
The data set used is " https://github.com/reisanar/datasets/blob/master/starbucks.csv"
